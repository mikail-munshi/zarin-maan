## 🚀 Deployed on Netlify (Free)

1. Go to :  (https://zarinimon.netlify.app/))


### Option B: GitHub + Netlify (More professional)

1. Create a free account at [github.com](https://github.com)
2. Create a new repository (e.g. `farewell-zarin`)
3. Upload all project files to the repo
4. Go to [netlify.com](https://netlify.com) → "Add new site" → "Import from Git"
5. Connect GitHub → Select your repo → Deploy
6. Done! Any future edits to GitHub auto-update the site.
